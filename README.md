# Synthesizer

Modular Synthesizer with MIDI support.

![Synth](screenshots/screenshot_1.png "Screenshot 1")

## Features
- Node based editor
- MIDI
  - Load MIDI files
  - Input from MIDI keyboards
  - Sequencer
- Synthesis
  - Subtractive
  - Additive
  - FM
- Sampler
- Effects
  - VA Filter
  - Delay
  - Flanger
  - Phaser
  - Distortion
  - BitCrusher
  - Equalizer
  - Compressor
  - Vocoder
- Visualization
  - Spectrogram
  - Oscilloscope
  - Decibel Meter
  - Vectorscope
- Save/load projects
